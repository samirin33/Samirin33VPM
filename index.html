Samirin33 VRC Utility(仮名)

Samirin33のVPMパッケージをここからゲットできます！

VCC にこのリポジトリを追加する方法

  1. VCC（VRChat Creator Companion）を開き、Settings を開く
  2. Packages タブをクリック
  3. Add Repository をクリック
  4. 表示された欄に、下の「Listing URL」を貼り付ける
  5. Add をクリックし、内容を確認して I Understand をクリック
  6. プロジェクトを開くと、追加したリポジトリのパッケージが表示されます

Listing URL

https://samirin33.github.io/Samirin33VPM/vpm.json
